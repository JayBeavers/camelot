A "node":http://nodejs.org wrapper for webcam controller providing configurable async frame grabbing. Grab frames, receive frame events!

h2. Requirements

This module assumes you have a working binary of fswebcam along with font management library and associated font path definition (e.g. GDFONTPATH )

h2. Examples

<pre>
var Camelot = require('./camelot.js');

var camelot = new Camelot( {
  'rotate' : '180',
  'flip' : 'v'
});

camelot.on('frame', function (image) {
  console.log('frame received!');
});

camelot.on('error', function (err) {
  console.log(err);
});

camelot.grab( {
  'title' : 'Camelot',
  'font' : 'Arial:24',
  'frequency' : 1    
});
</pre>

h2. Features

* Infinit frame grabbing based on frequency (in sec).
* Banner
* Png/Jpeg
* Rotation
* Resolution

h2. Installation
